How To Play



OVERVIEW

Sudoku puzzles are 9x9 grids that consist of only the numbers 1-9. We call each 1x1 box in this grid a cell. Every grid has exactly 81 cells. We tend to organize these cells into nine rows, nine columns, and nine 3x3 boxes as shown to the left (hover over the grid to see the rows/columns/boxes).


OBJECTIVE

The objective is to fill in the grid such that every row, column, and box contains the numbers 1 - 9 exactly once. For advanced solvers, time is another factor — solving Sudokus as fast as possible has not only interested puzzle-lovers but also become a worldwide competition!


GETTING STARTED

Sudokus are usually given with a number of cells filled in. These "clues" help solvers deduce the remaining numbers that go in the cells. Sudokus are logic puzzles, which means that every Sudoku can be solved without a single guess — all that is needed is time & wit.
